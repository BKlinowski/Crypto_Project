# Crypto Project

Website with bunch of crypto algorithms

## Implemented algorithms

### Ciphers
- AES128
- RSA
- Camellia
- Chacha20
- BlowFish

### Hashes
- SHA-256
- MD5

## Launching

### Using nodejs
`npm start`

### Or using docker

`docker pull bklinowski/crypto-project`  
`docker run --rm -d -p 3000:3000 bklinowski/crypto-project`
